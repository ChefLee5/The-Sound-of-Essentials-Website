# Data

Static structured content used by the React app belongs here.

## What Belongs Here

- Hero definitions in `heroes.json`
- Land metadata in `lands.json`
- Product metadata in `products.json`
- Track and gallery metadata
- Stable navigation or display data that is not translated text

## Naming

Use lowercase kebab-case or clear plural nouns.

```txt
heroes.json
lands.json
products.json
media-tracks.json
```

## What Does Not Belong Here

- React components
- CSS or visual styles
- API clients or secrets
- User-facing translated strings

Translated UI copy belongs in `src/i18n/locales/`.
